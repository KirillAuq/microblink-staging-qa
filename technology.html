About this audit

Full copy audit of the 13 technology/product pages against the “Web Redesign - Technology Pages” doc. Unlike the industry audit, no issue types are excluded. The doc holds several working drafts per page (V1/V2/“don't use this copy” blocks); where drafts compete, the latest non-deprecated version was treated as approved and the ambiguity is noted on the page. Expected template chrome (testimonials, blog feed, logo strips, footer) is excluded from findings.

Systemic patterns

Issues that recur across pages — fixing the source/template once likely resolves most instances.
Stat values differ from the doc
Most pages
Doc shows older figures (3.2B identities, 65,000+ data sources, page-specific 3rd stats); staging shows updated ones (4B+, etc.). Mostly the doc looks stale — but Sanctions “5,000+ data sources” (doc: 65,000+) and Workflow Builder's swapped pass-rate stat look like real errors.
Shared “Why Microblink” block overrides page copy
Automation, Command Center, Identity 360
Staging renders a shared platform block (“The Signal Layer, Owned End To End” / “Twelve years of computer vision…” / a Fraud Lab card) instead of each page's doc-specific Why-Microblink copy. Correct on Real-time Intelligence; layered on or replacing copy elsewhere.
CTA labels reworded
Real-time Intelligence + others
Doc's consistent “Explore [X]” / “Learn More About Identity Intelligence OS” CTAs are reworded on staging (“Learn more about…”, “Read more about…”, “Explore Identity Intelligence OS” dropping “the”).
Identity Document Verification built from a different draft
IDV page
Staging implements the earlier “Continuous Identity Document Verification” draft, not the doc's “Trust Every Identity” version — the RIVR metrics table and Try-the-Demo section are absent. Needs a content-owner decision on the canonical draft (the doc itself holds 3 overlapping drafts).
Agentic / AI-agent messaging dropped
Orchestration, Workflow Builder
Sentences about governing human + AI-agent flows were dropped from Orchestration (hero, a bullet, a Why-point) and Workflow Builder (handoff sentence) — weakening the agentic-commerce story vs the approved copy.
